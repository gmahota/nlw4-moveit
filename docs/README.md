
# Set Me up

```shell
  # Clone the repo
  $ git clone https://github.com/allanShady/nlw4-moveit.git
  
  # Navigate to the source folder
  $ cd source
  
  # Install dependences using yarn
  $ yarn install
```

# Enjoy it

```shell
  # Run 
  $ yarn dev 
```
> Then open the generated link with your prefered browser
